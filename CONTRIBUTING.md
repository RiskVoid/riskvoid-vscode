# Contributing to RiskVoid

Thank you for your interest in RiskVoid Security Linter!

## How to Contribute

### Reporting Bugs

Found a bug? Please [open an issue](https://github.com/riskvoid/riskvoid-vscode/issues/new?template=bug_report.md) with:

- VSCode version
- Extension version
- Steps to reproduce
- Expected vs actual behavior
- Error messages (if any)

### Requesting Features

Have an idea? [Open a feature request](https://github.com/riskvoid/riskvoid-vscode/issues/new?template=feature_request.md) describing:

- The problem you're trying to solve
- Your proposed solution
- Alternative approaches you've considered

### Questions & Discussions

For questions, ideas, or general discussion, use [GitHub Discussions](https://github.com/riskvoid/riskvoid-vscode/discussions).

## Code Contributions

RiskVoid is proprietary software. We do not accept pull requests at this time.

If you're interested in contributing code, please reach out to [hello@riskvoid.com](mailto:hello@riskvoid.com).

## Code of Conduct

Be respectful and constructive. We're all here to make security tools better.
